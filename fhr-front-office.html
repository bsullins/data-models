f_fhr_searches_daily
Searches performed in Mozilla products identified in the FHR submissions
snapshot_date date NOT NULL DEFO '1970-01-01'::date Date of snapshot (eg. ETL load date)
fhr_version varchar( 100 ) NOT NULL DEFO '#unknown' Version of FHR pings
search_date date NOT NULL DEFO '1970-01-01'::date date of search
product varchar( 255 ) NOT NULL DEFO '#unknown' Product used to perform search. ex: Firefox, Fennec, FirefoxOS
product_version varchar( 255 ) NOT NULL DEFO '#unknown' version of product used to perform search. ex: 21, 22, 23
channel varchar( 255 ) NOT NULL DEFO '#unknown' release channel of product. ex: release, aurora, nightly
locale varchar( 255 ) NOT NULL DEFO '#unknown' locale of installtion. ex: en-US
country varchar( 255 ) NOT NULL DEFO '#unknown' Country of installation. ex: United States of America, Spain, Poland
platform varchar( 255 ) NOT NULL DEFO '#unknown' platform of installation. ex: Windows, Android, iOS
platform_version varchar( 255 ) NOT NULL DEFO '#unknown' version of platform. ex: 7, 8, 95
partner varchar( 255 ) NOT NULL DEFO '#unknown' Search partner. ex: Google, Amazon, Bing, Lycos
search_location varchar( 255 ) NOT NULL DEFO '#unknown' Component of product used to perform search. ex: searchbar, urlbar
profile_count integer NOT NULL DEFO 0 count of profiles that peformed search. ex: 100, 10, 1000
search_count integer NOT NULL DEFO 0 count of searches. ex: 100, 1000, 10000


f_fhr_profile_age_buckets
Aggregated profile counts sourced from FHR pings grouped into buckets as defined by the Metrics team.
snapshot_date date DEFO '1970-01-01'::date Date snapshot was taken (eg. ETL Load Date)
fhr_version varchar( 100 ) NOT NULL DEFO '#unknown' Version of FHR ping
product varchar( 255 ) DEFO '#unknown' Product from which FHR ping was sourced. (eg. Firefox, Fennec)
product_version varchar( 255 ) DEFO '#unknown' Version of product (eg. 21, 22, 23)
channel varchar( 255 ) DEFO '#unknown' Release channel of product (eg. Aurora, Nightly, Beta, Release)
locale varchar( 255 ) DEFO '#unknown' Locale of installation (eg. en-US)
country varchar( 255 ) DEFO '#unknown' Country name from which FHR ping came (eg. Spain, Poland)
platform varchar( 255 ) DEFO '#unknown' Platform of installation (eg. Windows, Darwin, Android)
platform_version varchar( 255 ) DEFO '#unknown' Version of platform (eg. 4.3, 8, 7)
profile_age_buckets varchar( 255 ) DEFO '#unknown' Buckets for profile age: week 1 week 2 week 3 week 4 1 - 3 months 3 - 6 months 6 - 12 months 1 - 2 years 2 - 3 years 3+ years
profile_counts integer DEFO 0 Count of profiles

f_fhr_profile_age
This table contains 5 snapshots of FHR profiles and their age. This detailed level of data will most likely need to be aggregated via bucketing before analysis can be performed with an acceptable amount of performance (eg. response times).
snapshot_date date DEFO '1970-01-01'::date date of snapshot taken
fhr_version varchar( 100 ) NOT NULL DEFO '#unknown' Version of FHR pings
profile_create_date date DEFO '1970-01-01'::date Date profile was created
product varchar( 255 ) DEFO '#unknown' product of installation used by profile. ex: Firefox, Fennec, FirefoxOS
product_version varchar( 255 ) DEFO '#unknown' Version of product used by installation. ex: 21, 22, 23
channel varchar( 255 ) DEFO '#unknown' Release channel of installation. ex: Aurora, Nightly, Beta
locale varchar( 255 ) DEFO '#unknown' Locale of installation. ex: en-US
country varchar( 255 ) DEFO '#unknown' Country of installation. ex: United States of America, Spain, Brazil
platform varchar( 255 ) DEFO '#unknown' Platform of installation. ex: Windows, iOS, Android
platform_version varchar( 255 ) DEFO '#unknown' Version of platform. ex: 7, 8, 4.3
profile_age varchar( 255 ) DEFO '#unknown' Age of profile in days
profile_counts integer DEFO 0 Count of profiles
